## TCP x UDP

### UDP
- Conexão mais rápida
- Não confiavel
- Utilizado para câmeras de segurança e Live Stream

### TCP
- Conexão tem retorno
- Handshake
- Integridade, ordem dos dados
- Aplicativo de mensagens de texto
